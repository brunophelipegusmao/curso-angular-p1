Template Viarables

Template Variables são variáveis declaradas diretamente no HTML com o prefixo #, que referenciam um elemento DOM, componente, diretiva ou TemplateRef — permitindo acessar suas propriedades e métodos em outras partes do mesmo template:

HTML:

        <!-- referencia a um elemento DOM -->
    <input #taskInput placeholder="Digite a tarefa" />
    <button (click)="adicionarTarefa(taskInput.value)">Adicionar</button>

    <!-- referencia a um componente -->
    <app-form #meuForm />
    <button (click)="meuForm.submit()">Enviar</button>

TypeScript:

        export class AppComponent {
      adicionarTarefa(tarefa: string): void {
        console.log(tarefa);
      }
    }

A variável só é acessível dentro do template onde foi declarada e não está disponível no TypeScript do componente.

Outra abordagem para usarmos as Templates Variábles é:

HTML:

          <h2 #variable> Template Variables</h2>
    {{ variable.innerText }}

TypeScript:

           @ViewChild('h2')
     public nameH2!: ElementRef;
     ngAfterViewInit() {
       console.log(this.nameH2.nativeElement.innerText);
     }
