Deferrable Views

Deferrable Views é o mecanismo do Angular para carregar blocos do template de forma lazy — o conteúdo e seu JavaScript só são baixados quando necessário, reduzindo o bundle inicial e melhorando a performance. Usa o bloco @defer com blocos auxiliares para controlar cada estado do carregamento: 

         @defer (on viewport) {
        <app-comentarios />
      } @placeholder (minimum 1s) {
        <p>Placeholder antes de carregar</p>
      } @loading (minimum 1s; after 500ms) {
        <p>Carregando...</p>
      } @error {
        <p>Falha ao carregar.</p>
      }
  • @placeholder — exibido antes do carregamento iniciar
  • @loading — exibido durante o carregamento
  • @error — exibido se o carregamento falhar

Os gatilhos disponíveis são on viewport (quando o elemento entra na tela), on idle (quando o browser está ocioso), on interaction (quando o usuário interage) e when condition (quando uma expressão booleana for verdadeira).
