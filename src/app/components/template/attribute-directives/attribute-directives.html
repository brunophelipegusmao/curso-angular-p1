Diretivas de Atributo

Diretivas de Atributo são diretivas que modificam o comportamento ou a aparência de um elemento HTML, componente ou outra diretiva já existente — sem alterar a estrutura do DOM. São aplicadas como atributos no elemento-alvo usando o decorator @Directive.

Classes condicionais NgClass

Classes Condicionais é a sintaxe [class.nomeDaClasse] que adiciona ou remove uma classe CSS de um elemento com base em uma expressão booleana — a classe é aplicada quando a expressão é truthy e removida quando é falsy:

TypeScript:

              export class AppComponent {
          isAtivo = signal(true);
          isDestaque = signal(false);
    }

HTML:

         <ul [class.ativo]="isAtivo()" [class.destaque]="isDestaque()"></ul>

    <!-- multiplas classes via objeto -->
    <ul [class]="{ ativo: isAtivo(), destaque: isDestaque() }"></ul>

O Angular combina inteligentemente classes estáticas, bindings diretos e classes condicionais no mesmo elemento sem conflito.

Estilos dinâmicos NgStyle

Estilo dinâmico em diretivas de atributo é a capacidade de modificar propriedades CSS de um elemento diretamente pela diretiva, usando [style.propriedade] para um único estilo ou [style] com um objeto para múltiplos — sem necessidade de manipular o DOM manualmente:

TypeScript:

         @Directive({ selector: '[appDestaque]' })
    export class DestaqueDirective {
      @HostBinding('style.backgroundColor') bg = 'yellow';
      @HostBinding('style.color') color = 'black';
    }

HTML:

      <!-- via HostBinding na diretiva -->
    <p appDestaque>Texto destacado</p>

<!-- via binding direto no template -->
    <p [style.color]="corTexto">Texto colorido</p>
    <p [style]="{ color: corTexto, fontSize: tamanho }">Texto estilizado</p>

O Angular recomenda os bindings nativos [style] e [style.propriedade] no lugar da diretiva NgStyle por serem mais performáticos e não exigirem import adicional.
