Template

Um template é o HTML do componente que define sua estrutura visual. Ele estende o HTML padrão com sintaxe própria do Angular para exibir dados dinâmicos, reagir a eventos e controlar a renderização do DOM.

Text Interpolation

Text Interpolation é a sintaxe {{expressão}} que o Angular usa para renderizar valores dinâmicos de TypeScript diretamente no template HTML: 

            <p> {{usuario.nome}} </p>
      <p> {{2 + 2}} </p>
      <p> {{getTitulo()}} </p>

O Angular avalia a expressão, converte para string e atualiza o DOM automaticamente quando o valor muda.

Exemplo:

No template.ts ->

      export class TemplateBinding {
        public name: string = 'Bruno';
        public age: number = 38;
        public calculator(value1:number, value2:number) {
          return (
            value1 + value2
          )
        }
      }

No template.html ->


<p> Olá meu nome é {{ name }} e tenho {{ age }} anos.</p>


será reideriazado no navegador da seguinte forma:

Olá meu nome é {{ name }} e tenho {{ age }} anos.

Para a função calculator:


<p> A soma de 2 e 5 é {{ calculator(2, 5) }}</p>


será reideriazado no navegador da seguinte forma:

A soma de 2 e 5 é {{ calculator(2, 5) }}

Este recurso deve ser usado com cautela principalmente quando é usado com dados dinamicos

Além dos exemplos acima é possivel usar dentro do texto interpolation expressões lógicas e outras operações matemáticas, porém não é uma boa prática.

Property Binding

Property Binding é a sintaxe que permite definir dinamicamente propriedades de elementos HTML, componentes e diretivas a partir do TypeScript, usando colchetes [propriedade]: 

            <button [disabled]="!isFormValid()">Salvar</button>
      <img [src]="usuario.avatarUrl" />
      <custom-slider [value]="volume" />

Sempre que o valor no TypeScript muda, o Angular atualiza automaticamente a propriedade no DOM — sem necessidade de manipulação manual.

Exemplo:

-> desabilitado

No template.ts ->


public isDisabled = false;


No template.html ->


<button [disabled]="isDisabled"> Exemplo </button>

No template.ts ->


public isEnabled = true;


No template.html ->


<button [disabled]="isEnabled"> Exemplo </button>
